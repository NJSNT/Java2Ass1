# 黑胶唱片图书馆系统

## 项目概述

这是一个使用JavaFX实现的黑胶唱片图书馆管理系统，采用了MVVM、观察者模式和状态模式等设计模式。

## 设计模式实现

### 1. MVVM 模式
- **Model**: Vinyl, VinylLibrary - 数据模型层
- **ViewModel**: MainViewModel - 业务逻辑层
- **View**: MainController, main-view.fxml - 视图展示层

### 2. 观察者模式
- **Observable 接口**: Vinyl 和 VinylLibrary 实现
- **Observer 接口**: MainViewModel 实现
- **优点**: 数据变化时自动更新 GUI

### 3. 状态模式
- **VinylState 接口**: 定义状态行为
- **具体状态**:
  - AvailableState - 可用状态
  - ReservedState - 已预约状态
  - BorrowedState - 已借出状态
  - BorrowedReservedState - 已借出且已预约状态
- **优点**: 清晰的状态管理和转换

### 4. 单例模式
- **SessionManager**: 单例类，管理会话数据（当前用户ID、选中的黑胶ID）

### 5. 线程模式
- **UserSimulation**: 实现 Runnable 接口，模拟并发用户操作
- 在独立的守护线程中运行，执行随机操作

## 主要功能

1. 查看黑胶唱片列表（ID、标题、艺术家、发行年份、状态、借出人、预约人、是否标记为删除）
2. 预约黑胶唱片
3. 借出黑胶唱片
4. 归还黑胶唱片
5. 标记为删除
6. 添加新的黑胶唱片
7. 自动 GUI 更新（通过观察者模式）
8. 并发用户模拟

## 黑胶唱片状态

- **Available**: 可用（未被借出，无预约）
- **Reserved**: 已预约
- **Borrowed**: 已借出
- **Borrowed & Reserved**: 已借出且已被预约

## 构建和运行

### 要求
- Java 17 或更高版本
- Maven 3.6 或更高版本

### 构建
```bash
mvn clean install
```

### 运行
```bash
mvn javafx:run
```

## 使用说明

1. **设置用户 ID**: 在顶部文本框输入用户 ID
2. **选择黑胶**: 在表格中点击选择一个黑胶唱片
3. **执行操作**:
   - Reserve - 预约选中的黑胶
   - Borrow - 借出选中的黑胶
   - Return - 归还选中的黑胶
   - Mark for Removal - 标记为删除
4. **添加黑胶**: 点击 "Add Vinyl" 按钮添加新黑胶
